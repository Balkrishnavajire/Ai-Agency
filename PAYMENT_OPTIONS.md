# Payment Options Comparison

## Cheapest Options (Ranked by Fees)

### 1. **Manual Payment (UPI/Bank Transfer)** - **0% Fees** ✅ FREE
- **Fees:** ₹0 (completely free)
- **You receive:** ₹1,000 (full amount)
- **Pros:** No fees, full payment received
- **Cons:** Manual confirmation required
- **Best for:** Small volume, cost-conscious businesses

### 2. **PayU** - **1.5% Fees**
- **Fees:** 1.5% per transaction
- **You receive:** ₹985 (₹15 fee on ₹1,000)
- **Setup:** Free
- **Best for:** Lower transaction fees

### 3. **PayKun** - **1.75% Fees**
- **Fees:** 1.75% flat rate
- **You receive:** ₹982.50 (₹17.50 fee on ₹1,000)
- **Setup:** Free, no maintenance fees
- **Best for:** Simple pricing, easy integration

### 4. **Cashfree** - **1.75% Fees**
- **Fees:** Starting at 1.75%
- **You receive:** ₹982.50 (₹17.50 fee on ₹1,000)
- **Setup:** Free
- **Best for:** Instant refunds, next-day settlements

### 5. **Razorpay** - **2% Fees** (Current)
- **Fees:** 2% per transaction
- **You receive:** ₹980 (₹20 fee on ₹1,000)
- **Setup:** Free
- **Best for:** Most popular, great support

## Recommendation

**For maximum savings:** Use **Manual Payment (UPI/Bank Transfer)** - completely free!

**For automation:** Use **PayU** (1.5%) or **PayKun** (1.75%) for lower fees than Razorpay.

## Manual Payment Setup

The website now supports manual payment option where:
1. Customer books consultation
2. Customer sees your UPI ID / Bank details
3. Customer makes payment via UPI/Bank Transfer
4. You manually confirm payment (or use webhook if available)
5. System sends confirmation emails

This is **100% free** with **0% transaction fees**!
